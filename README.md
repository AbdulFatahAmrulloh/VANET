# 🛰️ **Simulasi Jaringan AODV di Jalan Jenderal Sudirman, Purwokerto - OMNeT++**  

Selamat datang di repositori ini! 🎉 Repositori ini berisi simulasi jaringan berbasis **OMNeT++** yang menggunakan **protokol AODV (Ad-hoc On-Demand Distance Vector)** untuk menganalisis kinerja jaringan di lingkungan nyata, yaitu **Jalan Jenderal Sudirman, Purwokerto**.  

## 📂 **Isi Repositori**  
✅ **Peta Jalan** – Representasi jaringan jalan di **Jalan Jenderal Sudirman, Purwokerto** dalam format yang sesuai untuk simulasi  
✅ **Konfigurasi Jaringan** – Implementasi protokol **AODV** untuk komunikasi antar node  
✅ **File Konfigurasi (omnetpp.ini)** – Pengaturan simulasi yang dapat disesuaikan  
✅ **Skrip Analisis** – Untuk mengevaluasi hasil simulasi  
✅ **Output Simulasi** – Data hasil pengukuran **End-to-End Delay, Throughput, dan Packet Delivery Ratio (PDR)**  

## 📊 **Parameter Evaluasi**  
🔹 **End-to-End Delay** – Waktu rata-rata yang dibutuhkan paket untuk mencapai tujuan  
🔹 **Throughput** – Jumlah data yang berhasil dikirim per satuan waktu  
🔹 **Packet Delivery Ratio (PDR)** – Persentase paket yang berhasil dikirim dibandingkan dengan yang dikirimkan  

## 🚀 **Cara Menjalankan Simulasi**  
1️⃣ Clone repositori ini ke lokal Anda:  
   ```bash
   git clone https://github.com/VANET/repo-name.git
   ```  
2️⃣ Buka **OMNeT++** dan import proyek ini  
3️⃣ Jalankan simulasi menggunakan file **omnetpp.ini**  
4️⃣ Analisis hasil simulasi dengan melihat output **End-to-End Delay, Throughput, dan PDR**  

## 🌍 **Tujuan dan Manfaat**  
🔹 Memahami performa protokol **AODV** dalam skenario dunia nyata  
🔹 Menganalisis kualitas jaringan dalam **lingkungan perkotaan**  
🔹 Berguna untuk penelitian di bidang **VANET (Vehicular Ad-Hoc Networks)** atau **MANET (Mobile Ad-Hoc Networks)**
